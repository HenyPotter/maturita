# Strukturované programování - sekvence, I/O

## Sekvence
Nejjednodušší typ algoritmu. Skládá se pouze z mezních značek, značka pro vstup avýstup dat a značek pro zpracování (přiřazení) dat

### Povolená operace
- součet (+)
- rozdíl (-)
- součin (*)
- mocnina SQR (^)
- logický součet (OR)
- logický součin (AND)
- negace (NOT)

### Nepovolené operace
- podíl (/) - nějaký
